## PR 규칙
- PR제목은 [n주차]이름으로 보내주시면 됩니다.
  `ex) [2주차]서희원`

- PR내에 내용은 없어도 됩니다.
- 만약 PR을 한번 더 보내야 한다면 (PR제목)+로 보내주시면 됩니다. `ex)[2주차]서희원+`
